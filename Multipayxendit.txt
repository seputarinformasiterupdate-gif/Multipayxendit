1. Buat akun GitHub
- Buka https://github.com
- Daftar akun baru
- Verifikasi email
2. Buat Repository Baru
- Klik New Repository
- Nama: multipayxendit-billing
- Pilih Public
- Centang Add README
- Klik Create Repository

7. Buat Tagihan
Input:
- Nama Customer
- Nomor WhatsApp
- Nominal
- Keterangan
- Tanggal Jatuh Tempo
Contoh:
Nama : Budi Santoso
WA : 08123456789
Nominal : Rp150.000
Keterangan : Pembayaran Tagihan
8. Generate Invoice
Format:
MPX-20260530-001
MPX-20260530-002

NAMA APLIKASI:
MultipayXendit Billing System
SLOGAN:
"Satu Platform untuk Semua Tagihan dan Pembayaran"
DESKRIPSI:
MultipayXendit Billing System adalah aplikasi penagihan online yang memungkinkan admin membuat invoice, menghasilkan link pembayaran unik untuk setiap pelanggan, mengelola status pembayaran, mengunggah bukti transfer, dan menyimpan data secara online menggunakan Firebase serta di-hosting melalui GitHub Pages.
